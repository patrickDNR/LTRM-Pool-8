# LTRM-Pool-8
Hold data analysis scripts for Pool 8 to produce useful graphs and prep for DNR annual report

# Fish Data Prep Script  
This script take raw output from LTRM download (https://umesc.usgs.gov/data_library/fisheries/fish1_query.shtml) and processes in order to calculate Catch Per Unit Effort and community diversity metrics. 

# hydrology annual report LD8
This script uses dicharge data from Lock & Dam 8 near Genoa, WI to summarize hydrology data to give insight on trends in flow and high discharge events.
